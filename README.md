# HR Leave Management System

A full-stack HR Leave Management System built with NestJS, Next.js, TypeScript, PostgreSQL, Supabase, Prisma, and JWT authentication.

## Tech Stack

### Backend

- NestJS
- TypeScript
- Prisma ORM
- PostgreSQL / Supabase
- JWT Authentication

### Frontend

- Next.js
- TypeScript
- Tailwind CSS

## Core Features

- Role-based access control: admin, manager, employee
- Employee profile management
- Leave type management
- Leave balance tracking
- Leave request CRUD
- Manager approval workflow
- Dashboard
- Calendar view
- Reports
- Demo seed data
- Production deployment
